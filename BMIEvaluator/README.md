# BMI Evaluator ⚖️

### Description
A health-utility tool that calculates the Body Mass Index (BMI) based on weight and height inputs. It provides an immediate health status assessment based on standard medical categories.

### Features
* **User Input Processing**: Captures weight (kg) and height (m) dynamically.
* **Health Categorization**: Automatically determines if the user is underweight, normal weight, or overweight.
* **Precision**: Uses floating-point arithmetic for accurate results.

### How to Run
Run the `BMIEvaluator.java` file from your terminal or IDE.
