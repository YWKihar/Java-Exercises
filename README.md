# Java Programming Exercises 🚀

This repository contains a collection of Java programs designed to practice fundamental programming concepts, including logic building, mathematical operations, and conditional statements.

## 📋 Table of Contents

| Project Name | Description |
| :--- | :--- |
| **RectangleArea** | Calculates the area and perimeter of a rectangle. |
| **BMIEvaluator** | Computes Body Mass Index (BMI) and provides weight status. |
| **GreatestNumberFinder** | Identifies the largest among a set of input numbers. |
| **TriangleValidator** | Determines if three given sides can form a valid triangle. |
| **CurrencyConverter** | Simple tool to convert between different currency values. |
| **OddEvenChecker** | Checks if a given integer is odd or even. |
