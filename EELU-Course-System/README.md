# EELU Course System 🎓

### Description
The **EELU Course System** is a Java-based management application designed to handle student data and academic course structures. This project demonstrates a solid understanding of Object-Oriented Programming (OOP) principles, including encapsulation and structured data management within a professional project hierarchy.

### Key Features
* **Structured Packages**: Organizes code into logical components for better maintainability.
* **Student Management**: Core logic to handle student records and course associations.
* **NetBeans Integration**: Fully configured with `nbproject` settings for immediate deployment.
* **Clean Architecture**: Separation of logic and execution via a dedicated main class.

### Prerequisites
* **Java Development Kit (JDK)**: Version 8 or higher.
* **IDE**: NetBeans, IntelliJ IDEA, or Eclipse.

### How to Use
1. Clone or download the project folder.
2. Open the project in your preferred IDE.
3. Run the `EELU.java` file located in the `src/eelu/` directory.
