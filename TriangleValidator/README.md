# Triangle Validator 📐

### Description
A logic-based program that validates whether three given side lengths can form a valid triangle. It also identifies the specific type of triangle based on the inputs.

### Features
* **Geometric Logic**: Checks the "Triangle Inequality Theorem."
* **Classification**: Identifies if the triangle is Equilateral, Isosceles, or Scalene.
* **Input Validation**: Ensures only positive numerical values are processed.

### How to Run
Execute `TriangleValidator.java` and enter the side lengths when prompted.
